結論

backdrop-filter が参照できるもの:
- CSS filter functions (blur, saturate, brightness, contrast, etc.)
- SVG filter via url(#filter-id)

SVG feImage が参照できるもの:
- 外部画像 (PNG, data:URI)
- 同一SVG内の他のSVG要素 (#id)
- ❌ HTML要素は参照不可

つまり、CSS gradientをdisplacement mapとして使うには:
1. Canvas経由でdata:URIに変換 → 計算コスト発生
2. SVG linearGradient + feColorMatrix → ブレンドモード制限あり
