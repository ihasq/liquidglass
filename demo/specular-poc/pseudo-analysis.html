Dual Rim Specular Analysis

Why Dual Rim?

Real glass and glossy surfaces reflect light from multiple directions. A single light source creates an unnatural, one-sided highlight. By using two opposing light sources (e.g., 2 o'clock + 8 o'clock), we simulate:

  • Ambient light reflection - Light bouncing from environment
  • Key + fill lighting - Standard photography/3D lighting setup
  • Natural glass appearance - Real glass reflects from multiple angles

Dual Rim Algorithm

// Two light directions (e.g., 315deg + 135deg = 2h + 8h) light1 = { cos(315deg), sin(315deg) } // Upper-right light2 = { cos(135deg), sin(135deg) } // Lower-left // For each edge pixel: dot1 = normal . light1 // How much this edge faces light 1 dot2 = normal . light2 // How much this edge faces light 2 lightFacing1 = max(0, dot1) // Clamp negative (backfacing) lightFacing2 = max(0, dot2) * weight // Secondary light can be weaker // Combine both lights (additive) combinedLight = min(1, lightFacing1 + lightFacing2) // Apply to ring pattern if (edgeDist in [1, 1+ringWidth]): alpha = sin(ringPos * PI) * combinedLight * intensity

Dual Rim Variants

Dual Rim (Equal) Best

Both lights at 100% strength. Creates symmetric highlight on opposing edges.

weight = 1.0 combinedLight = light1 + light2

Best for: Centered UI elements, symmetric panels

Dual Rim (Weighted) Flexible

Primary light stronger than secondary. Creates subtle asymmetry.

weight = 0.6 // Secondary at 60% combinedLight = light1 + light2 * weight

Best for: Directional scenes, depth emphasis

Dual Rim + Soft Fresnel Premium

Adds Fresnel falloff layer under the rim. Richest appearance.

fresnel = F0 + (1-F0) * (1-t)^3 rim = dualRim(light1, light2) alpha = fresnel * 0.3 + rim * 0.9

Best for: Hero elements, premium UI

Quad Rim (4 lights)

Four lights at corners. Most uniform but can look flat.

lights = [315, 45, 135, 225] degrees combinedLight = sum(max(0, dot(n, L))) * 0.5

Best for: Circular elements, uniform glow

Recommendation: 315deg + 135deg (2h + 8h)

This diagonal configuration provides the most natural appearance because:

  • Matches typical studio lighting (key light upper-right, fill lower-left)
  • Creates visual interest without being distracting
  • Works well with both portrait and landscape orientations
  • Secondary light at 60% strength adds depth without overpowering

Clock Position to Degrees

// Clock positions map to angles (12 o'clock = 270deg, 3 o'clock = 0deg) function clockToAngle(hour) { return (90 - hour * 30 + 360) % 360; } // Examples: // 2 o'clock = 90 - 60 = 30... wait, let's recalculate // Actually: 12h=270, 3h=0, 6h=90, 9h=180 // So 2h = 270 + 60 = 330... or more simply: // angle = (450 - hour * 30) % 360 2 o'clock -> 315deg (upper-right) 8 o'clock -> 135deg (lower-left) 12 o'clock -> 270deg (top) 6 o'clock -> 90deg (bottom)

Combining with Environment-Aware

Dual-rim can be combined with environment-aware methods:

// Base: environment luminance extraction envLuma = extractLuminance(background) // Overlay: dual-rim static specular dualRim = generateDualRim(light1, light2) // Combine with weighted blend finalSpecular = envLuma * 0.4 + dualRim * 0.6 // This ensures visibility on dark backgrounds while // still reflecting bright elements from background

Final Recommendation

Use: Dual Rim + Soft Fresnel

Configuration:

  • Light 1: 315deg (2 o'clock) at 100%
  • Light 2: 135deg (8 o'clock) at 60%
  • Ring width: 2% of min(width, height)
  • Fresnel blend: 30% Fresnel + 90% rim

This provides the most natural glass appearance with proper edge highlighting from two directions, plus a subtle Fresnel glow for depth.
