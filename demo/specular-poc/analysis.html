Filter Pipeline

Recommendation

Best Overall: Method 3 (Fresnel Hybrid)

For production use, Method 3: Fresnel Hybrid offers the best balance:

  • Physical accuracy: Schlick Fresnel approximation is industry standard
  • Environment awareness: Background luminance directly affects specular intensity
  • Reasonable cost: Only 1.4x overhead vs static method
  • Natural appearance: Grazing angle enhancement matches real glass behavior

For maximum quality regardless of cost, Method 4: Multi-stage IBL provides the most realistic result with roughness-based reflection sharpness.

For direction-aware reflection without Fresnel complexity, Method 2: Directional Blur provides pseudo-environment mapping using displacement.

Implementation Priority

1. Method 3: Fresnel Hybrid Recommended for v1
2. Method 1: Luminance Extract Fallback option
3. Method 2: Directional Blur Optional enhancement
4. Method 4: Multi-stage IBL High-end option
5. Method 5: feSpecularLighting Experimental
